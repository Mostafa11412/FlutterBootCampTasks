# 📱 Device Info Task  

A Flutter application that retrieves and displays device information such as model, OS type, and OS version.  

## 🚀 Features  
- Detects whether the device is running Android or iOS.  
- Retrieves and displays the device model and OS version.  

## 📂 Project Structure  
- **`main.dart`** → Entry point of the app, initializes `MyApp`.  
- **`home_screen.dart`** → UI displaying device information.  
- **`device_info_manager.dart`** → Handles device info retrieval using `device_info_plus`.  

## 🛠️ Dependencies  
- [device_info_plus](https://pub.dev/packages/device_info_plus)
